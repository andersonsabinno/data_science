# Ciências de dados na industria

A ciência de dados transforma um conjunto de dados dispersos em decisões, nesse caso, pega esses dados e encontra padrões e pode também fazer previsões.

## Etapas da ciências de dados

1. <b>Entender o problema<br></b>
    Basicamente, se você não entende não vai conseguir fazer uma análise correta.

2. <b>Coleta de dados<br></b>
    Depois de entender o problema, agora devemos reunir os dados, por exemplo:<br>
        - histórico de paradas e falhas;<br>
        - dados de produção por turnos;<br>
        - etc
    Esses dados podem está em diversas fontes(planilhas, banco de dados,  etc)

3. <b>Preparação dos dados<br></b>
    Aqui temos que organizar, padronizar e limpar os dados para que seja possível fazer uma análise deles corretamente, tudo isso incluir verificar os formatos das datas, limpar códigos duplicados, etc.

4. <b>Análise Exploratória<br></b>
    Aqui que começamos fazer a análise com os dados organizados, aqui começamos fazer algumas perguntas para entender o cenário: Qual máquina apresenta mais problemas? Em que turnos ocorre mais falhas?
    Para buscar respostas são criados gráficos e efetuados cálculos.

5. <b>Apresentação de dados<br></b>
    É hora de comunicar as descobertas com gráficos informando o que foi análisado.

6. <b>Tomada de decisão<br></b>
    Com os dados analisados e apresentados a equipe pode tomar decisões.

## Ferramentas ou linguagens

Quais ferramentas iremos utilizar no processo da análise dos dados:

1. Planilhas 
2. Python
3. Pandas (Biblioteca de Python)
4. Power BI/ Google Data Studio 

## Aplicação da ciência de dados

<b>Eficiência Energética</b> <br>
<b>Objetivo: </b> reduzir consumo de energia.<br>
Fazendo a analise dos dados de consumo por máquina e por horário, encontra-se um aumento anormal de energia nas madrugadas.<br>
<b>Resultado: </b> Problema estava em um compressor que permanecia ligado sem necessidade.
<br>

Claro que poderíamos citar várias situações que podemos aplicar a ciência de dados.
