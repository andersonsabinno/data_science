# Dados

Dado é a matéria prima da informação, somente com os dados brutos não conseguirmos analisa-los. Logo, se eu agrupo esses dados e coloco em um contexto isso se torna uma informação.

Exemplo de dados: 5 
Exemplo de informação: Alice tem 5 anos. 

A diferença entre dados estruturados é que podemos achá-los em planilhas, csv, etc. Enquanto que os dados não estruturados são imagens, vídeos, etc.

Antes de fazer a análise dos dados precisamos executar o processo ETL (extrair, transformar e carregar), basicamente, precisamos coletar os dados brutos, realizar ajustes e organizar as informações para analises.
